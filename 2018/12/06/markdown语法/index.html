markdown语法

什么是Markdown：

  Markdown是一种纯文本格式的标记语言。通过简单的标记语法，它可以使普通文本内容具有一定的格式。

标题：

  在想要设置为标题的文字前面加来#表示
一个#是一级标题，二个#是二级标题，以此类推。支持六级标题。

段落：

  要创建段落,你需要回车一行或多行。不能使用空格或者制表符。
注：缩进段落两个汉字只需需段落前加入两个&emsp;即可。

  这是缩进了的一个段落。
这是未缩进的一个段落。

加粗文本：

  你只需在文本的两侧添加**或者__即可。

  加粗了。

斜体：

  你可以在文本两侧添加*或者_即可。

  我是斜体字。

**注：你也可以嵌套的使用加粗与斜体字:

既变粗又变斜

引用文字：

  要引用文字，你可以在文字之前添加>即可。

这是一个引用。

这也是一个引用。

引用段落：

  要引用段落，你可以在段落之间的空行处添加>即可。

这是一个段落的引用

这是一个段落的引用
这是一个段落的引用

嵌套引用文字：

  要嵌套引用的话你可以在段落之前添加>>按顺序依次嵌套。

这是一个段落的引用

这是另一个段落的引用

有序列表：

  创建有序列表，你需要在列表项之前添加数字后跟英文的句号
注：你可以缩进列表以形成嵌套

  1. 有序列表
    1. 有序列表
  2. 有序列表

无序列表：

  创建无序列表，你需要在列表项之前添加*或-或+
注：你可以缩进列表(两个空格)以形成嵌套

  • 无序列表

    • 无序列表

  • 无序列表

你可以将列表与文字引用一起嵌套：

  • 无序列表

    引用

  • 无序列表

    引用

图片：

  ![Alt text](url "optional title"
Alt text：用来描述图片的关键词址。”optional title”：鼠标悬置于图片上的提示文字。

突出代码：

  少量代码你可以这样:`content`

content

  代码块你可以这样
``` codename
  content
```

分割线：

  要使用分割线你可以使用***或---或___。

链接：

  引用链接你可以[name](url “showinfo”)‘showinfo’是鼠标悬停在链接上显示的信息

转义字符：

  在你需要转义的符号前面加入\即可

`code`
*变粗*
_变斜体_
