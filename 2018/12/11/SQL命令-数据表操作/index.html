SQL是什么

SQL 是用于访问和处理数据库的标准的计算机语言。

创建数据表 Create Table TableName

创建数据表是应注意以下几方面：

  1. 表中包含了那些数据项，即有多少列。
  2. 指定列的名称，名称应该有其具体的含义。
  3. 根据列所保存数据的性质，指定每一列的数据类型。
  4. 指定那些列可以为空值(null)，那些不能为空值。
  5. 指定数据完整性约束。
  6. 指定索引以及索引的类型。
  7. 指定数据的安区配置和访问权限。

语法格式

列级约束(ColumnConstraint)

非空约束，主键约束，外键约束，唯一性约束，检查约束，默认约束

表级约束(TableConstrint)

主键约束，外键约束，唯一性约束，检查约束，默认约束(同列级)

删除数据表

删除数据表是应注意以下几点

  1. 数据表正在使用时，无法删除表。

  2. 当数据表是一个父表，且子表存在时，父表不能删除，必须先删除字表中对父表的外键约束，才能删除父表。

  3. 当存在主外键关系时，删除表与创建表的顺序刚好相反:创建时，应先创建父表后创建子表；删除时，应先删除子表，后删除父表。

  4. 删除表将删除与该表相关的所有数据，结构定义，约束，索引等，且删除后无法恢复，因此执行时该操作时候应十分慎重，平时应该注意备份数据库。
